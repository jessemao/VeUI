# VweUI

> WeUI for vue 2.0.
> 使用 Vue.js 2.0 为微信 WeUI(1.0+) 库进行量身设计、封装。目前所有组件均已封装完毕，将会在接下来一段时间陆续更新使用文档，发布至NPM registry。

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
